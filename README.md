# LRU Virtual Memory Management Simulator

## Overview

This project simulates virtual memory management using the Least Recently Used (LRU) page replacement algorithm. It demonstrates how pages are loaded into memory, replaced, and managed efficiently when page faults occur.

## Features

- Simulation of virtual memory management
- LRU Page Replacement Algorithm
- Page Fault Detection
- Memory Frame Management
- Console-based visualization
- Object-Oriented Programming implementation in C++

## Technologies Used

- C++
- Object-Oriented Programming
- STL (Vector, hash maps, etc.)
